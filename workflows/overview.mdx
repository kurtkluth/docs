## How to Use a Template

<Steps>
  <Step title="Find the template">
    Locate the template you want in the Free Resources channel or this knowledge base.
  </Step>
  <Step title="Download or copy the workflow">
    Download the JSON file or copy the workflow configuration from the resource page.
  </Step>
  <Step title="Import into n8n">
    Open n8n and import the workflow via **Workflows → Import from File**.
  </Step>
  <Step title="Update credentials">
    Replace the placeholder credentials — API keys and account connections — in each node.
  </Step>
  <Step title="Test before activating">
    Run the workflow with a manual trigger and verify each step completes correctly before switching it on.
  </Step>
</Steps>

<Note>
  All templates in AI Builders Circle are built and tested by Josh. They're starting points — customize them for your specific tools and business.
</Note>
